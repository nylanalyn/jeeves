# Jeeves IRC Butler

Jeeves is a modular IRC companion written in Python. The bot connects to IRC networks, loads feature modules from `modules/`, and persists shared state in `config/`. A quest-oriented web dashboard lives under `web/quest/`.

- Read the full documentation: `docs/README.md`
- Contributor guidelines: `docs/AGENTS.md`
- Theme reference: `docs/themes.md`
- Roadmap and design notes: `docs/roadmap.md`

Install dependencies with `pip install -r requirements.txt`, configure the bot via `config/config.yaml`, and start it using `python3 jeeves.py`.
